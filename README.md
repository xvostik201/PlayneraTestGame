# Тестовое задание — Playnera (Junior Unity Developer)

Реализация механики нанесения макияжа по ТЗ компании Playnera.

---

## О проекте

Полноценная реализация игровой механики **Makeup System** согласно требованиям тестового задания Playnera.

**Основные реализованные механики:**
- Drag-управление рукой
- Подлёт руки к инструменту
- Dipping-анимация (окунание кисти/помады в цвет)
- Промежуточная позиция руки на уровне груди
- Progress-наполнение макияжа (fillAmount)
- Автоматическое завершение нанесения при 100%
- Отдельная apply-анимация для крема
- Автоматическое стирание макияжа спонжем (Loofan)
- Плавное затухание (fade) слоёв макияжа + частицы
- Возврат кистей обратно в палитру после использования

---

## Особенности реализации

- **Переключатель режима Loofan**  
  В `HandController.cs` есть публичный bool `_useSmoothLoofanAnimation`.  
  По умолчанию **выключен** (`false`) — спонж работает строго по ТЗ (автоматически, без драга и анимации пути).

  - Используется **DOTween** для всех анимаций
  - Поддержка **UIParticle** (ParticleEffectForUGUI)
